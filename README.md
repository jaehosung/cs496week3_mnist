## Requirements
- Python 3
- Tensorflow > 0.12
- Numpy
- Django (2, 0, 1, 'final', 0)


## Training
```bash
python train.py
```


## Run Server
```bash
python manage.py runserver 0.0.0.0:8000
```

### Deploy to Heroku ###
    $ heroku apps:create [NAME]
    $ git push heroku master
